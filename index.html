General Questions

Er det vanskelig å kode?

Det kan virke vanskelig til å beynne med, men etterhvert som man lærer mer koding blir det lettere å lettere, så lenge man gir litt insats.

Er det vanskelig å lære programering?

Det er vanskeligere for noen enn andre men det er ikke umulig, hvis du tar deg tid til å lære det så kommer du til å gjøre det.

Hva er HTML og Javascript?

HTML eller HyperText Markup Language er koden som står for å vise ting på nettsider, som skrift. JavaScript er et kodespråk som kjører i de fleste nettlesere. Det brukes for interaktivitet og for å kjøre kommandoer på nettsider.

Hvordan lærer man JS, HTML og CSS effektivt?

Søk på YT, bruk StackOverFlow, få tilgang til github copilot.

Hvordan få Dark mode i vscode

Man kan enkelt laste ned themes i Extensions tab-en, men det er allerede et theme kalt dark+ som man kan aktivere under settings > themes

Hvordan lager man en nettside?

En nettside er bare HTML kode. For at det skal være noe fancy eller opp til dagens standard så må man kombinere HTML kode med andre kodespråk.

Hvilken pc / laptop er best for koding?

Bruk det du vil, men Mac er mindre supported og lærere bruker det ikke. Mange andre i bransjen bruker Mac selv om. Trenger ikke en spesiell type laptop for å kode.
